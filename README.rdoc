= Coaches and Players

Application uses {SL messaging API}[http://privatelabelsms.com/api/#api-messaging] for sending SMS messages from coaches to players and receiving replies.

== Usage

1. clone repository
2. replace <b>[API KEY GOES HERE]</b> in <i>demo.rb</i> with your ServiceLayer[http://servicelayer.recessmobile.com/] App's API key
2. run <b>ruby demo.rb</b>
3. open http://yourhost.com:4567 in browser
4. in order to test replies specify http://yourhost.com:4567/gateway as <b>Gateway URL</b> for your ServiceLayer[http://servicelayer.recessmobile.com/] App
